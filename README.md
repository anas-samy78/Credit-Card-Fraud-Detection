# 💳 Credit Card Fraud Detection System

An end-to-end Machine Learning project for detecting fraudulent credit card transactions using **XGBoost**, featuring an interactive **Streamlit** web application for real-time predictions.

---

## 📌 Project Overview

Credit card fraud is one of the most challenging problems in the financial industry due to the highly imbalanced nature of transaction data.

This project focuses on building a robust fraud detection system capable of identifying suspicious transactions while minimizing false alarms through feature engineering, model optimization, and interactive deployment.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* XGBoost Classification Model
* Model Performance Evaluation
* Feature Importance Visualization
* Interactive Streamlit Dashboard
* Professional Project Report
